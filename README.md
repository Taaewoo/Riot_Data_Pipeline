## Git clone
~~~
$ git clone https://github.com/Taaewoo/Riot_Data_Pipeline.git
~~~

## Riot API Key 파일 생성
~~~
$ vim src/main/resources/riotApiKey.properties
~~~
"riotApiKey.properties" 파일에 아래 내용을 본인 Riot API 넣어서 작성
~~~
riot.api.key = RGAPI-xxxxx-xxxxx-xxxx
~~~
